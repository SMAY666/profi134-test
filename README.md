# Тестовое задание PROFI 134
## Инструкция по запуску проекта
1. Создайте в корне проекта .env файл следующего содержания:
````
HOST=producer
PORT=8001
RMQ_URL=amqp://admin:admin@rabbitmq:5672
BOT_TOKEN=YOUR_TG_BOT_TOKEN
````
2. Находясь в корневой директории выполните команду ``docker-compose up -d --build``
3. Перейдите в диалог с ботом, токен которого вы указали в ``BOT_TOKEN`` и отправьте ему сообщение с любым содержанием. Это нужно, чтобы бот мог присылать вам сообщения. <br>
*Примечание*: Бот отправляет сообщение в последний активный чат.

## Отправка сообщений
Для отправки сообщений отправьте запрос:
``curl --location --request POST 'http://localhost:8001/event' \
--header 'Content-Type: application/json' \
--data-raw '{
    "Test": "123",
    "Field": 4,
    "obj": {"heal": 20}
}'``, <br> где ``--data-raw (Body)`` может быть объектом с любыми полями.

## Логи
Для просмотра логов используйте команду ``docker logs <container_name>``
